# Deep Graph Anomaly Detection with Contrastive Learning

Unsupervised anomaly detection on graph-structured data using Graph Autoencoders (GAE)
and SimCLR-inspired contrastive learning on the PROTEINS benchmark dataset.

This project is a direct prototype of the anomaly detection framework proposed for
[ML4SCI GENIE — GSoC 2025](https://github.com/felipetp-ctrl/gsoc-ml4sci-genie-submission).

## Results

| Model | Approach | Test AUC |
|-------|----------|----------|
| GAE | Reconstruction-based | 0.434 |
| **SimCLR** | **Contrastive learning** | **0.709** |

SimCLR outperforms the reconstruction baseline by **+0.29 AUC**, demonstrating that
learning discriminative graph representations via contrastive objectives is more
effective than adjacency reconstruction for anomaly detection on the PROTEINS dataset.

## Overview

The project implements two unsupervised anomaly detection approaches for graphs:

**Graph Autoencoder (GAE) — baseline**
- GNN encoder produces node-level embeddings
- Decoder reconstructs the adjacency matrix via inner product
- Anomaly score = MSE reconstruction error per graph

**SimCLR on Graphs — main model**
- Two augmented views of each graph are generated via node feature dropout and edge dropout
- A GNN encoder is trained with NT-Xent contrastive loss to produce similar embeddings
  for views of the same graph and dissimilar embeddings for different graphs
- Anomaly score = Euclidean distance from the embedding to the centroid of the
  training distribution (following Luo et al., 2022)

- Binary labels: enzyme (1) vs non-enzyme (0)

## Architecture

### GNN Encoder
- 2 EdgeConv layers (hidden dim = 64) with BatchNorm
- Global mean + max pooling concatenated → 128-dim graph embedding
- Same architecture used in the ML4SCI GSoC submission (Common Task 2)

### SimCLR Model
- GNN Encoder + Projection Head (128-dim → 128-dim MLP with BatchNorm)
- NT-Xent loss with temperature τ = 0.5
- Graph augmentations: node feature dropout (p=0.1) + edge dropout (p=0.1)

### GAE
- GNN Encoder (node-level, no pooling)
- Decoder: sigmoid(Z @ Z.T) per graph
- BCE loss on adjacency reconstruction

## Connection to GSoC Project

This project is a direct implementation of the contrastive anomaly detection framework
proposed for **ML4SCI GENIE — Deep Graph Anomaly Detection with Contrastive Learning**.

In the full GSoC project, the same pipeline will be applied to particle physics data:
- Dataset: LHC Olympics 2020 (dijet resonance signals)
- Encoder trained on QCD background jets only
- Anomaly score identifies potential new physics events

The PROTEINS benchmark here validates the pipeline before extending it to HEP data.
